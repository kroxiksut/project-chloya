# Методология CHLOYA — русская версия

Русская версия — первоисточник методологии CHLOYA. Методология находится в разработке и будет публиковаться здесь отдельными тематическими главами.

> **Следующая итерация:** `0.3.1`. Она разрабатывается и утверждается по главам.

## Короткая формула

> Человек задаёт направление и принимает решения. ИИ организует и выполняет работу в ограниченных полномочиях. Код остаётся общим понятным языком. Контекст локализуется и получает метки доверия. Контракты связывают части. Проектная память принадлежит проекту, а готовность подтверждается проверяемыми доказательствами.

## Краткое определение

CHLOYA — методология управления созданием и развитием программного обеспечения, в которой человек определяет цели, участвует в постановке и декомпозиции задач, утверждает значимые решения и результаты, а временные ИИ-исполнители выполняют работу в пределах ограниченного, переносимого, снабжённого метками доверия и проверяемого контекста.
