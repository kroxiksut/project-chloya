# История изменений CHLOYA

Этот журнал кратко фиксирует существенные изменения методологии между итерациями. Он не заменяет историю Git и не включает обычные редакторские правки.

## 0.3.1 — в разработке

### Реализовано

- Создано русское и английское оглавление, рабочая структура глав и их статусы для итерации `0.3.1`.
- Русская версия закреплена как первоисточник; английское содержание ведётся как рабочий перевод.
- Уточнены краткое определение CHLOYA и раскрытие названия: Constrained Handoffs, Local Ownership и Yield Assurance.
- Подготовлена русская глава 32 с единым списком из 39 источников, сохранённой нумерацией, ссылками и датами обращения.
- Опубликован английский перевод главы 32 с сохранением общей нумерации, оригинальных названий источников, ссылок и дат обращения.

Содержательные изменения будут фиксироваться здесь по мере разработки и согласования глав версии `0.3.1`.
